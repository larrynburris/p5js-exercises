P5.js Mouse Interaction

The properties mouseX and mouseY represent the current position of the mouse in the canvas. Move the mouse around the canvas to see how the mouse is tracked and the coordinates change.

The property MouseIsPressed is a boolean that allows you to know if the user is holding the mouse button down. Notice how we darken the background when holding the mouse button down.

Defining the mousedPressed() function in the sketch file will act as an event handler for mouse presses. See how mouse clicks capture the current mouse position to be drawn as points in future draw() executions.
